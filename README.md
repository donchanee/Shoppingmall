# Springboot-shoppingmall
여성 의류 쇼핑몰 프로젝트

## 1. 기획 의도
- 스프링 부트를 기반으로 여성 의류 판매 쇼핑몰 웹사이트 구축
- 기획부터 배포까지 실제 운영함에 있어 관리자를 위한 기능 설계

## 2. 사용 기술

### 백엔드
- Java 11
- Spring Boot Security
- Spring Boot
- Spring AOP
- Spring MVC
- JUnit5
- JPA
- Lombok
- Querydsl

### Database
- MySqL

### AWS
- EC2

### Build tool
- gradle

### 프론트엔드
- JavaScript
- JQuery
- Thymeleaf
- Ajax

### 형상관리
- GitHub

### 기술목록

- 페이징
- 트랜잭션
- GIT
